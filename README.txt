# La Marmite Bleue - HACCP

## Fonctionnalités
- Import Excel
- Sauvegarde Firebase
- Chargement Firebase
- Compatible GitHub Pages
- Base HACCP évolutive

## Installation Firebase

1. Créer un projet sur :
https://firebase.google.com/

2. Activer Firestore Database

3. Copier les informations Firebase

4. Remplacer dans index.html :

const firebaseConfig = {
  apiKey: "...",
  authDomain: "...",
  projectId: "..."
}

## Mise en ligne GitHub

1. Créer un repository GitHub
2. Importer index.html
3. Activer GitHub Pages
4. Le site sera accessible en ligne
